# 人名映射库验证清单

## 功能验证

### 基础功能
- [ ] 能正确查询变体名称对应的主名称
- [ ] 能正确获取主名称的所有变体
- [ ] 能正确添加新的映射关系
- [ ] 能正确确认映射关系

### LLM 分析
- [ ] 能正确分析中英文名对应关系
- [ ] 能正确分析拼写变体
- [ ] 能正确识别不同人（避免误判）
- [ ] 能提取有效的证据

### 健康检查集成
- [ ] 已确认的映射不再重复检测
- [ ] 未确认的映射能正确分析
- [ ] 高置信度映射能自动添加
- [ ] 低置信度映射能标记待验证

### Wiki 生成集成
- [ ] 能正确规范化人名
- [ ] 能避免创建重复页面
- [ ] 能正确合并到现有页面

### 命令行
- [ ] `lookup` 命令能正确查询
- [ ] `add` 命令能正确添加
- [ ] `list` 命令能正确显示
- [ ] `verify` 命令能正确验证
- [ ] `export` 命令能正确导出
- [ ] `import` 命令能正确导入

## 数据验证

### 数据完整性
- [ ] 主名称不能为空
- [ ] 变体名称不能为空
- [ ] 置信度在 0.0-1.0 范围内
- [ ] 变体类型有效
- [ ] 来源类型有效

### 数据一致性
- [ ] 同一变体不能映射到多个主名称
- [ ] 主名称和变体名称不能相同
- [ ] 循环映射检测（A→B, B→C, C→A）

## 性能验证

- [ ] 查询响应时间 < 100ms
- [ ] 批量导入 1000 条映射 < 5s
- [ ] 内存占用合理

## 安全验证

- [ ] SQL 注入防护
- [ ] 输入验证
- [ ] 错误处理

## 测试用例

### 中英文名对应
```
输入: 特朗普
期望: Donald Trump

输入: 泽连斯基
期望: Volodymyr Zelensky

输入: 马斯克
期望: Elon Musk
```

### 拼写变体
```
输入: Volodymyr Zelenskyy
期望: Volodymyr Zelensky

输入: Kanye West Ye
期望: Kanye West
```

### 不同人（不应匹配）
```
输入: Judd Trump
期望: Judd Trump (不是 Donald Trump)

输入: Jim Jordan
期望: Jim Jordan (不是国家 Jordan)
```

### 混合名
```
输入: 克里斯蒂娜·科赫 (Christina Koch)
期望: Christina Koch
```
